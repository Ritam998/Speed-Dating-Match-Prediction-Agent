# 🧠 Speed Dating Match Prediction Agent

Predict whether two participants in a speed dating event are likely to be a match using machine learning.

This repository contains an end-to-end AI system that trains, evaluates, and deploys a matchmaking model based on socio-demographic and preference data collected from speed-dating experiments.

---

## 🚀 Project Summary

Speed dating produces rich interaction data. This project builds a **predictive model** to estimate whether two people will mutually select each other (`match = yes`) based on their attributes and preferences.
